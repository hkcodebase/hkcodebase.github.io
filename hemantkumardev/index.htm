  • Netflixdata Analyzer (analyze.netflixdata.com)
    This data analytics application, developed using Java, Spring Boot, Thymeleaf, Docker, and Docker Compose, provides insights on your watched content on Netflix. Users can upload their ViewingActivity.csv file from the Netflix-provided watched history in zip format for analysis. The project is available on GitHub.
